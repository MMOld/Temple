Packagetemple.data.flashvars
Classpublic class FlashVars
InheritanceFlashVars Object

This class is a Singleton wrapper around the flashvars, so they can be accessed at places where there is no Stage.

You have the possibility to set a default and a class-type for each flashvar individually.

In combination with a FlashVarNames enum class you know which flashvars are used in the application.

configurateVar()method
public static function configurateVar(name:String, defaultValue:Class = null, type:* = null):void

Use this to configurate the flashvars with a default value and a type.

Parameters

name:String — The flashvar name (use FlashVarNames.NAME)
 
defaultValue:Class (default = null) — The defaultValue if the flashvar does not exists or is empty
 
type:* (default = null) — The class to cast this FlashVar to. Array will use split(',').


Throws
TempleArgumentError — When defaultValue is not of type
 
TempleError — When not initialized

Example 
         FlashVars.configurateVar(FlashVarNames.LANGUAGE, 'nl', String);
         FlashVars.configurateVar(FlashVarNames.VERSION, 1, int);
         FlashVars.configurateVar(FlashVarNames.IS_DEMO, true, Boolean);

getValue()method 
public static function getValue(name:String):*

Returns the flashvar value. If the flashvar is empty or not given it will return the defaultValue. If a class is given it will be casted to that class.

If name is not in the flashvar-pool, a warning is logged and an empty String is returned

Parameters

name:String — The flashvar name (use FlashVarNames.NAME)

Returns
*

Throws
TempleError — When not initialized

Example 
         FlashVars.getValue(FlashVarNames.LANGUAGE);

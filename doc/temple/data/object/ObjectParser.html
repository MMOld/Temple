Packagetemple.data.object
Classpublic class ObjectParser
InheritanceObjectParser Object

The ObjectParser parses an Object (ig JSON) to an other object (ig a typed DataValueObject). The objects to parse to must be of type IObjectParsable. The ObjectParser calles the 'parseObject' method on the IObjectParsable and passes the Object that needs to be parsed.

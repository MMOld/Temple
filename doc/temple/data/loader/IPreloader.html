Packagetemple.data.loader
Interfacepublic interface IPreloader extends IDisplayObject , IDestructable , IEventDispatcher , IBitmapDrawable

An interface used by visual preloaders. All classes that implement IPreloadable can use IPreloader object as their preloader.

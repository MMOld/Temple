defaultDebugModeproperty
public static var defaultDebugMode:int = 2

When debug is not set in the url, this debugmode is used for the DebugManager. Possible values are:

DebugMode ConstantDescription
DebugMode.NONEDebug will be set to false on all debuggable object. So no debug messages will be logged.
DebugMode.CUSTOMDebug can be set to each debuggable object individually.
DebugMode.ALLDebug will be set to true on all debuggable object. So debug messages will be logged

See also

temple.debug.DebugManager

registerObjectsproperty 
public static var registerObjects:Boolean = true

When set to true, all Temple objects are registered in the Memory class with a weak reference. Usefull for debugging to track all existing objects. After destructing an object (and force a Garbage Collection) the object should no longer exist. NOTE: Changing this value must be done here or before the super() inside the constructor of your application root

See also

temple.debug.Registry

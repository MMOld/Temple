Interfacepublic interface IDebuggable
Implementors DebugManager, NotificationCenter, SiteDisabler

Interface for objects that can debugged. The interface adds a 'debug' property. If this value is set to true, debug information will be logged.

IDebuggable objects can be handled by the DebugManager

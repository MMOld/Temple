Classpublic class ZoomBehavior
InheritanceZoomBehavior BoundsBehavior AbstractDisplayObjectBehavior AbstractBehavior CoreEventDispatcher flash.events.EventDispatcher

The ZoomBehavior makes it possible to zoom in or out on a DisplayObject. The ZoomBehavior uses the decorator pattern, so you won't have to change the code of the DisplayObject. The DisplayObject can be zoomed by using code or the MouseWheel.

Method Detail
destruct()method
override public function destruct():void

Destroys the objects, all intern listeners will be removed.

When overriding this method, always call super.destruct() at the end!

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.
When a Temple object is destructed a DestructEvent.DESTRUCT is dispatched from the object (if the object implements IDestructableEventDispatcher).

Note: Bear in mind that it is possible an object can be destructed more than once.

Examples
     new ZoomBehavior(mcClip);

If you want to limit the zooming to a specific bounds, you can add a Rectangle. By adding the Reactangle you won't be able to zoom the DisplayObject outside the Rectangle:

     new ZoomBehavior(mcClip, new Reactangle(100, 100, 200, 200);

It is not nessessary to store a reference to the ZoomBehavior since the ZoomBehavior is automaticly destructed if the DisplayObject is destructed.

Packagetemple.ui
Interfacepublic interface IDisplayObject extends IEventDispatcher , IBitmapDrawable

Interface that contains all the properties of a DisplayObject. This interface is implemented by all DisplayObjects of the Temple, but not by Flash native DisplayObects. This interface is extended by other interface to force they can only be implemented by DisplayObjects.

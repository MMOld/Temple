FrameDelay()Constructor
public function FrameDelay(callback:Function, frameCount:int = 1, params:Array = null)

Creates a new FrameDelay. Starts the delay immediately.

Parameters
callback:Function — the callback function to be called when done waiting
 
frameCount:int (default = 1) — the number of frames to wait; when left out, or set to 1 or 0, one frame is waited
 
params:Array (default = null) — list of parameters to pass to the callback function
Method Detail
destruct()method
override public function destruct():void

Release reference to creating object. Use this to remove a FrameDelay object that is still running when the creating object will be removed.

Examples
     new FrameDelay(init);
To execute function 'init' after 10 frames, use: 
     new FrameDelay(init, 10);
To call function 'setProps' with parameters, executed after 1 frame: 
     new FrameDelay(setProps, 1, [shape, 'alpha', 0]);
     
     private function setProps (shape:Shape, property:String, value:Number):void
